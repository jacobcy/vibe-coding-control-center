# Roadmap JSON Standard (v3) - 已废弃

**状态**: ⚠️ **已废弃 (Deprecated)**

**废弃原因**: v3 已转向使用 **GitHub Labels** 和 **Milestone** 管理 roadmap，不再使用本地 `roadmap.json` 文件。

**替代方案**:
- Roadmap 规划: 使用 `roadmap/*` 标签 → [github-labels-standard.md](../github-labels-standard.md)
- Roadmap 管理: 使用 GitHub Milestone → [roadmap-label-management.md](../roadmap-label-management.md)
- 优先级管理: 使用 `priority/*` 标签 → [github-labels-standard.md](../github-labels-standard.md)
- 状态管理: 使用 `state/*` 标签 → [github-labels-standard.md](../github-labels-standard.md)
