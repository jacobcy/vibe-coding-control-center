Vibe Coding Control Center

Usage: vibe [options] [command]

Core Commands:
  chat           Start AI assistant
  flow           Development workflow
  init           Initialize new project

Configuration & Environment:
  config         Manage configuration files
  env            Manage environment variables
  check          System diagnostics & health check
  alias          Manage command aliases
  equip          Install & update tools

Options:
  -g, --global   Use global installation (e.g., vibe -g flow status)
  -h, --help     Show this help message
  -v, --version  Show version information

Run 'vibe' without arguments for interactive mode.
